UPDATE AUTHORS DATA

CEDIA: REPOSITORIO SEMÁNTICO DE INVESTIGADORES DEL ECUADOR

This module allows automatic extraction information regarding authors, and update this data to marmotta platform.

DEFAULT SOURCE

  • NAME: "CEDIA"
  • PROVIDER: "SPARQL"
  • SPARQL Endpoint: "http://190.15.141.102:8890/sparql"
  • PREFIX: "http://190.15.141.102:8080/dspace/"

ACTUALIZAR REPOSITORIO DE AUTORES

Se cargara la informacion de los autores provenientes desde las Diferentes Fuentes (Endpoints) que esten activas. Las uris de los recursos (Autores) que hayan sido cargadas exitosamente a la plataforma (LOD Marmotta) seran registrados en un archivo en la siguiente ruta: {marmotta-home}/listAuthorsUpdate_{NombreEndpoint}.aut.

"Authors-Module" depende de "Linked Data Caching-Module" que es el modulo que provee la funcionalidad de administracion de los Endpoints.
