Los datos de los autores provienen de la plataforma DSPACE de cada una de las Universidades miembros de CEDIA. En un proceso previo, el grupo de repositorios con el proyecto "Aplicación de Linked Open Data a Repositorios CEDIA", realizó el trabajo de transformar la información de DSPACE a formato RDF siguiendo las normas y principios de Linked Data. Uno de los resultados de este proyecto fue la generacion de repositorios RDF de cada una de las fuentes analizadas. En este proyecto, para extraer la información de autores, se accede a los repositorios RDF antes mencionados a través del servicio SPARQL ENDPOINT de cada una de las fuentes.

La información de las publicaciones de los autores se extrae desde diferentes fuentes de datos. Una fuente de datos de publicaciones es una interfaz que expone datos acerca de artículos (incluyendo los autores, abstracts, keywords, citas, etc.) indexada en dicha fuente.
